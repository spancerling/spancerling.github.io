css实现居中的小tips

1 水平居中


1-1 行内元素
设置text-align: center
1-2 定宽块状元素
设置左右margin值为auto

2 垂直居中


2-1 父元素高度确定的单行文本
设置line-height的值为height;

3 水平和垂直居中(常见div)


3-1 知道宽高的情况下采用绝对定位的方式，兼容基本浏览器
.one { position: absolute; width: 200px; height: 200px; top: 50%; left: 50%; margin-top: -100px; margin-left: -100px; }
3-2 行内元素居中比如文字
.one { width: 100px; height: 100px; line-height: 100px; text-align: center; }
3-3 移动端常使用css3的transform:translate(x,y)属性
.one { position: absolute; top: 50%; left: 50%; transform: translate(-50%,-50%); -webkit-transform: translate(-50%,-50%); -moz-transform: translate(-50%,-50%); -ms-transform: translate(-50%,-50%); }
3-4 使用calc 支持Firefox 4 及更高版本浏览器，对于更早版本的浏览器，需要添加前缀，不支持IE8。
.one { width: 100px; height: 100px; position: absolute; top: calc(50% - 50px); left: calc(50% - 50px); }
